# vol-cli

> Get and set sound volume


## Install

```
$ npm install --global vol-cli
```


## Usage

<img width="283" src="screenshot.png">

```
$ vol --help

  Usage
    $ vol (Interactive CLI)
    $ vol <level>
    $ vol mute

  Examples
    $ vol 65
```

If encountering any problems in Windows, use the `volume` alias.
